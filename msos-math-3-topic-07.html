Lesson 1: Picture Graphs

A picture graph uses symbols (each symbol = a number of items). Always check the KEY! If ☆ = 2 books and one row shows 4 stars, that row = 8 books.

📝 Practice Worksheet

Try 6 problems. Type each answer, then click Check.

1. If ★ = 2 and 3 stars are shown, count = ?
2. If ★ = 5 and 4 stars, count = ?
3. If ★ = 3 and 6 stars, count = ?
4. If ★ = 4 and 5 stars, count = ?
5. If ★ = 2 and 7 stars, count = ?
6. If ★ = 10 and 3 stars, count = ?

Lesson 2: Bar Graphs

Bar graphs use rectangles to show amounts. The taller the bar, the bigger the number. Always read the SCALE on the side. Compare bars by subtracting.

📝 Practice Worksheet

Try 6 problems. Type each answer, then click Check.

1. If red = 8 and blue = 5, red − blue = ?
2. If A = 12 and B = 7, A + B = ?
3. If C = 15 and D = 9, difference = ?
4. Total of 4, 6, 8 = ?
5. Total of 7, 3, 5 = ?
6. Difference of 20 and 12 = ?

Lesson 3: Line Plots

A line plot uses Xs above a number line to show how often each value appears. Handy for measurements (like plant heights, shoe sizes). Count Xs to find the mode.

📝 Practice Worksheet

Try 6 problems. Type each answer, then click Check.

1. If 5 Xs are at value 3, total value = ?
2. If 3 Xs at value 4 and 2 Xs at value 2, sum = ?
3. Xs at values 6, 6, 8. Sum = ?
4. Xs at 2, 4, 6, 8. Sum = ?
5. Xs at 3, 3, 3. Sum = ?
6. Xs at 5, 5, 10. Sum = ?
